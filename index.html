BACKTRACK

This event is all about reverse coding. In this event participants will be given executable code files / compiled code written in the language opted by them. The event requires thinking in the reverse order of program development. The participants have to develop a code that generates the same output as the given executable file. It is a puzzle to figure out the underlying code.
Languages Applicable: C, Java

C OrDEr

This event is all about rearranging to code snippets. Participants will be given different code snippets. The event requires the participants to fetch necessary snippets and rearrange them to form a logical program
Languages Applicable: C

TECH QUIZ

This event is about testing the general knowledge in the field of computer science. Participants will be given 25-30 questions based on common topics related to computer science.

CODE GURU

This event is to test the knowledge of programming language. Event consists of 2 rounds. In round 1 participant has to act as a compiler to dry run the code snippets given. Participants selected for the second round will be arranged in a team of 2 where they will be given 2-3 problems to code. Participant taking least time to code will win.
Languages applicable: C, Java, C#

WEB CRAFT

This event is all about testing the creativity of an individual. Participants will be provided a theme to design a web page based on that theme. Certain html components will be asked to necessarily include in the design. Only HTML, CSS, JavaScript are allowed. Use of JQuery is not allowed.

TYPING TEST

This short event is about testing the typing speed of the individuals on a keyboard

DATA BOSS

Participants will be given a database with 2-3 tables. Some result sets from the database will be provided and the participants have to figure out the SQL query for the given result sets.
